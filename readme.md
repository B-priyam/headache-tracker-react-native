## ğŸ“Œ Overview

**Headache Tracker** is a fully featured mobile application created to help users record, track, visualize, and analyze headache episodes with exceptional detail.

Built with **React Native + Expo**, this app provides a **clean, modern UI** that makes logging headaches effortless while providing professional-grade insights and downloadable reports suitable for medical use.

The goal is to help users:

- Understand patterns
- Monitor trends over time
- Identify triggers
- Improve communication with doctors
- Take control of their headache and migraine management
